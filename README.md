# realtime-client-server-demo
C++ real-time client-server networking demo (TCP &amp; UDP) with performance-focused architecture.

## Quick Test Results (Localhost)
- TCP line-based protocol (HELLO / PING-PONG)
- RTT (localhost): ~0–1 ms (avg ~0.4–0.5 ms)
- Tested with 50+ sequential client connections successfully
